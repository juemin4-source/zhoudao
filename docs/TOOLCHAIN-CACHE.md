# 工具链缓存策略 v0.0.10

> 本规范描述周道工具链的缓存规则。
> LSP 与 CLI 不得因为缓存产生不同语义结果。

---

## 1. 模块编译缓存

### 1.1 缓存键

周道模块编译缓存的键由以下组成：

- 源码文件内容哈希
- 周道编译器版本
- 项目配置摘要
- 依赖模块缓存键摘要

### 1.2 失效条件

- 源码文件修改
- 编译器版本变化
- 项目配置变化
- 依赖模块重新编译

---

## 2. Python 元数据缓存

### 2.1 缓存键

- Python 解释器身份（路径 + 版本）
- 模块名和版本

### 2.2 失效条件

- 解释器变化
- 模块版本变化
- 用户显式清除缓存

---

## 3. 一致性规则

- LSP 与 CLI 不得因为缓存产生不同语义结果
- 清除缓存后结果必须一致
- 不得缓存用户程序的运行时副作用
- 缓存失效规则必须有文档和测试
