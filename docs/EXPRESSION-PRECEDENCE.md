## 1. 优先级总表

完整优先级从最低到最高（1 = 最低绑定力，7 = 最高绑定力）：

```
等级 7 (原子)      : 字面量、标识符、函数调用、括号分组
等级 6 (前缀)      : 负号（一元减）、并非（逻辑非）
等级 5 (乘除)      : 乘  除  整除  余
等级 4 (加减)      : 加  减
等级 3 (成员/比较)  : 在…中  不在…中  等于  不等于  大于  小于  不少于  不多于  就是  不是
等级 2 (逻辑且)    : 且
等级 1 (逻辑或)    : 或
```

### 1.1 快速参考

| 优先级 | 类别 | 算符 | 结合性 |
|--------|------|------|--------|
| 1（最低） | 逻辑或 | 或 | 左 |
| 2 | 逻辑且 | 且 | 左 |
| 3 | 比较/成员 | 等于/不等于/大于/小于/不少于/不多于/在…中/不在…中/就是/不是 | 左 |
| 4 | 加减 | 加/减 | 左 |
| 5 | 乘除 | 乘/除/整除/余 | 左 |
| 6 | 前缀 | 负号/并非 | 右（前缀单目） |
| 7（最高） | 原子 | 数字/文本/列表/标识符/函数调用/括号 | — |

## 2. 各优先级详解

### 2.1 等级 1：逻辑或

```
语法: 或表达式 := 且表达式 ( "或" 且表达式 )*
示例: 甲大于5或甲小于3  → (甲大于5) 或 (甲小于3)
结合: 左结合
```

### 2.2 等级 2：逻辑且

```
语法: 且表达式 := 非表达式 ( "且" 非表达式 )*
示例: 甲大于5且甲小于10 → (甲大于5) 且 (甲小于10)
结合: 左结合

注意: 且 优先于 或。 a或b且c → a或(b且c)
```

### 2.3 等级 3：比较与成员

#### 2.3.1 比较算符

```
语法: 比较表达式 := 成员表达式 (比较算符 成员表达式)?
比较算符: 等于 | 不等于 | 大于 | 小于 | 不少于 | 不多于
示例: 甲加3大于乙减2  → (甲加3) 大于 (乙减2)
结合: 不结合（不能链式比较）
```

注意：周道 **不支持** 类似 Python 的链式比较（`a < b < c`）。

```
错误: 如果甲小于乙小于丙      → 语法错误
改写: 如果甲小于乙且乙小于丙  → 正确
```

#### 2.3.2 成员算符

```
语法: 成员表达式 := 加性表达式 ( "在" 加性表达式 "中" )?
                     | 加性表达式 ( "不在" 加性表达式 "中" )?
示例: 甲在名单中  → 成员判断
      甲不在名单中 → 非成员判断
```

注意：「在…中」/「不在…中」是后缀算符，其作用域覆盖整个括号/字符串。

#### 2.3.3 身份算符

```
就是 | 不是（甲不是...本身）
示例: 甲就是乙  → 身份判断
      甲不是乙本身 → 否定身份判断
优先级: 与比较算符同级
```

### 2.4 等级 4：加减

```
语法: 加性表达式 := 乘性表达式 ( "加" 乘性表达式 | "减" 乘性表达式 )*
示例: 3加4乘5 → 3加(4乘5)  — 乘性 优先于 加性
结合: 左结合
```

### 2.5 等级 5：乘除

```
语法: 乘性表达式 := 前缀表达式 ( "乘" 前缀表达式 | "除" 前缀表达式 | "整除" 前缀表达式 | "余" 前缀表达式 )*
示例: 3乘4减5 → (3乘4)减5  — 乘性 优先于 加性
结合: 左结合
```

### 2.6 等级 6：前缀

```
语法: 前缀表达式 := "减" 原子      — 一元负号
                    | "并非" "(" 条件 ")"  — 逻辑非
                    | 原子
示例: 减3       → 一元负号应用于 3
      减3加4    → (减3)加4，不是 减(3加4)
      并非(条件) → 逻辑非
```

注意：逻辑非 `并非` 必须跟随括号 `（条件）`，不能直接放在表达式前。

```
正确: 并非(甲等于乙)
错误: 并非甲等于乙   → 缺少括号
```

### 2.7 等级 7：原子

```
语法: 原子 := 数字 | 文本 | 列表 | 布尔字面量 | 空值 | 标识符 | 函数调用 | "(" 条件 ")"
示例: 42, 【文本】, ［1、2、3］, 真, 假, 没有值, 变量名, 函数(参数)
```

## 3. 完整优先级示例

### 3.1 算术

```
输入: 3加4乘5
解析: 3加(4乘5) = 3 + 20 = 23
不是: (3加4)乘5 = 35
```

```
输入: 10减3乘2加4除2
解析: 10减(3乘2)加(4除2) = 10-6+2 = 6
```

```
输入: 减3乘减4
解析: (减3)乘(减4) = (-3)*(-4) = 12
```

### 3.2 逻辑

```
输入: 甲大于5或甲小于3且甲不等于0
解析: 甲大于5 或 (甲小于3 且 甲不等于0)
```

```
输入: 并非(甲等于乙或丙等于丁)
解析: 并非( (甲等于乙) 或 (丙等于丁) )
```

### 3.3 混合

```
输入: 甲加3乘2大于乙减1
解析: (甲加(3乘2)) 大于 (乙减1)
```

```
输入: 甲在名单中或乙在名单中
解析: (甲在名单中) 或 (乙在名单中)
```

## 4. 括号显式指定

用户可以使用全角括号 `（……）` 显式改变优先级：

```
3加4乘5         → 3加(4乘5) = 23   (默认优先级)
(3加4)乘5       → (3加4)乘5 = 35   (括号覆盖)
```

```
并非(甲等于乙)    → 逻辑非   (强制)
并非(甲)等于乙    → 错误: 「并非」后括号必须包含完整条件
```

## 5. 结合性细节

| 算符类别 | 结合性 | 含义 |
|---------|--------|------|
| 二元算术算符 | 左结合 | `a加b减c` → `(a加b)减c` |
| 二元逻辑算符 | 左结合 | `a且b且c` → `(a且b)且c` |
| 前缀算符 | 右结合（前缀单目） | `减减3` → `减(减3)` |
| 比较算符 | 不结合 | `a大于b大于c` → 错误 |
| 成员算符 | 不结合 | `a在b在c中` → 错误 |
| 赋值类 | 不结合 | 周道中赋值不是表达式 |

## 6. 优先级相关歧义

### 6.1 常见误解

| 表达式 | 误解 | 正确解析 |
|--------|------|---------|
| `3加4乘5` | `(3加4)乘5=35` | `3加(4乘5)=23` |
| `甲大于乙或丙小于丁` | `甲大于(乙或丙)小于丁` | `(甲大于乙)或(丙小于丁)` |
| `减3加4` | `减(3加4)=-7` | `(减3)加4=1` |

### 6.2 消歧建议

当用户不确定优先级时，建议使用括号显式指定：

```
3加(4乘5)          ← 明确
(甲大于乙)或(丙小于丁)  ← 明确
```

## 7. 优先级表（PEG 语法形式）

```
条件        := 或表达式
或表达式    := 且表达式 ("或" 且表达式)*
且表达式    := 非表达式 ("且" 非表达式)*
非表达式    := "并非" "（" 条件 "）" | 比较表达式
比较表达式  := 成员表达式 (比较算符 成员表达式)? (身份算符 成员表达式)? (空值后缀)?
成员表达式  := 加性表达式 (成员后缀)?
加性表达式  := 乘性表达式 (加性算符 乘性表达式)*
乘性表达式  := 前缀表达式 (乘性算符 前缀表达式)*
前缀表达式  := "减" 原子 | 原子
原子        := 数字 | 文本 | 列表 | 真 | 假 | 没有值 | 标识符 | 函数调用 | "（" 条件 "）"

比较算符    := "等于" | "不等于" | "大于" | "小于" | "不少于" | "不多于"
身份算符    := "就是" | "不是"
成员后缀    := "在" 加性表达式 "中" | "不在" 加性表达式 "中"
空值后缀    := "没有值"
加性算符    := "加" | "减"
乘性算符    := "乘" | "除" | "整除" | "余"
```

## 8. 冻结清单

| # | 规则 | 说明 |
|---|------|------|
| 1 | 优先级 7 > 6 > 5 > 4 > 3 > 2 > 1 | 不可变更顺序 |
| 2 | 乘除优先于加减 | 同数学标准 |
| 3 | 且优先于或 | 同标准逻辑 |
| 4 | 比较算符不链式 | 不接受 `a < b < c` |
| 5 | 前缀算符右结合 | 一元算符 |
| 6 | 二元算符左结合 | 算术和逻辑 |
| 7 | 括号可覆盖优先级 | 全角 `（` `）` |
| 8 | `并非` 必须带括号 | 不同于通用写法 |
