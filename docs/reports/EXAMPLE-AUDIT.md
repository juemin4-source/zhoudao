## 1. 倒计数

### 源码

```
设数量为3，当数量大于0时，一直显示数量，并使数量减1。
```

### 句法分解

```
句子
├── 顶层表达式: 绑定
│   ├── 设
│   ├── 标识符: 数量
│   └── 表达式 (原子): 3
├── ，(句内分隔)
├── 顶层表达式: 当句
│   ├── 当
│   ├── 条件: 比较表达式
│   │   ├── 标识符: 数量
│   │   ├── 比较算符: 大于
│   │   └── 原子: 0
│   ├── 时，
│   ├── 一直
│   └── 动作链
│       ├── 动作: 输出
│       │   └── 表达式 (标识符): 数量
│       ├── ，
│       └── 动作: 算术变更
│           ├── 使
│           ├── 标识符: 数量
│           ├── 算术算符: 减
│           └── 表达式 (原子): 1
└── 。
```

### AST 结构

```
Sentence
├── Stmt: Bind(name="数量", value=IntLiteral(3))
├── Stmt: While(
│   condition=BinOp(left=Var("数量"), op="大于", right=IntLiteral(0)),
│   body=[
│       Stmt: Print(value=Var("数量")),
│       Stmt: AugAssign(target="数量", op="-", value=IntLiteral(1))
│   ]
│ )
```

### 歧义评估

无。`当…时，一直…` 有明确边界：`时，` 终止条件，`一直` 标记体开始。体到 `。` 结束。

### Python 输出

```python
数量 = 3
while 数量 > 0:
    print(数量)
    数量 -= 1
```

## 2. 年龄分支

### 源码

```
设年龄为16，如果年龄不少于18，就显示【成年】，不然，如果年龄不少于12，就显示【少年】，不然就显示【儿童】。
```

### 句法分解

```
句子
├── 顶层表达式: 绑定
│   ├── 设
│   ├── 标识符: 年龄
│   └── 表达式 (原子): 16
├── ，
├── 顶层表达式: 如果句
│   ├── 如果
│   ├── 条件: 比较表达式
│   │   ├── 标识符: 年龄
│   │   ├── 比较算符: 不少于
│   │   └── 原子: 18
│   ├── ，就
│   ├── 动作: 输出
│   │   └── 文本: 【成年】
│   ├── ，不然，如果
│   ├── 条件: 比较表达式
│   │   ├── 标识符: 年龄
│   │   ├── 比较算符: 不少于
│   │   └── 原子: 12
│   ├── ，就
│   ├── 动作: 输出
│   │   └── 文本: 【少年】
│   ├── ，不然就
│   └── 动作: 输出
│       └── 文本: 【儿童】
└── 。
```

### AST 结构

```
Sentence
├── Stmt: Bind(name="年龄", value=IntLiteral(16))
├── Stmt: If(
│   condition=BinOp(left=Var("年龄"), op=">=", right=IntLiteral(18)),

│       (condition=BinOp(left=Var("年龄"), op=">=", right=IntLiteral(12)),

│   ],
│   else=[Stmt: Print(value=StrLiteral("儿童"))]
│ )
```

### 歧义评估

无。`不然，如果` 和 `不然就` 有明确关键字边界。

### Python 输出

```python
年龄 = 16
if 年龄 >= 18:
    print("成年")
elif 年龄 >= 12:
    print("少年")
else:
    print("儿童")
```

## 5. 单表达式函数

### 源码

```
设平方（数）为数乘数，设结果为平方（5），显示结果。
```

### 句法分解

```
句子
├── 顶层表达式: 单式函数
│   ├── 设
│   ├── 标识符: 平方
│   ├── （
│   ├── 形参: 数
│   ├── ）
│   ├── 为
│   └── 表达式: 乘性表达式
│       ├── 标识符: 数
│       ├── 乘
│       └── 标识符: 数
├── ，
├── 顶层表达式: 绑定
│   ├── 设
│   ├── 标识符: 结果
│   ├── 为
│   └── 表达式: 函数调用
│       ├── 标识符: 平方
│       ├── （
│       ├── 参数: 5
│       └── ）
├── ，
├── 顶层表达式: 输出
│   └── 标识符: 结果
└── 。
```

### AST 结构

```
Sentence
├── Stmt: FuncDef(
│   name="平方",
│   params=["数"],
│   body_type="single",
│   body=BinOp(left=Var("数"), op="*", right=Var("数"))
│ )
├── Stmt: Bind(name="结果", value=Call(
│   func=Var("平方"),
│   args=[IntLiteral(5)]
│ ))
├── Stmt: Print(value=Var("结果"))
```

### 歧义评估

无。`设标识符（…）为` 是函数定义 vs `设为` + 函数调用：`设平方（5）为结果` 会是函数调用后在绑定。解析器通过 `为` 前是 `（参数）` 还是仅 `（形参）` 区分——实际上 `设平方（数）为` 中形参位置是单个标识符，而函数调用中的参数可以是任意表达式。但 `设为平方（5）为结果` 中 `平方（5）` 是调用表达式，然后 `为结果` 绑定——不混淆。

### Python 输出

```python
def 平方(数):
    return 数 * 数
结果 = 平方(5)
print(结果)
```

## 6. 多分支函数与所得

### 源码

```
设绝对值（数），如果数不少于0，就以数为所得，不然就以0减数为所得。
```

### 句法分解

```
句子
├── 顶层表达式: 复式函数
│   ├── 设
│   ├── 标识符: 绝对值
│   ├── （
│   ├── 形参: 数
│   ├── ）
│   ├── ，
│   └── 动作链（函数体）:
│       └── 动作: 如果句
│           ├── 如果
│           ├── 条件: 比较表达式
│           │   ├── 标识符: 数
│           │   ├── 比较算符: 不少于
│           │   └── 原子: 0
│           ├── ，就
│           ├── 动作: 以…为所得
│           │   └── 表达式 (标识符): 数
│           ├── ，
│           ├── 不然就
│           └── 动作: 以…为所得
│               └── 表达式: 减性表达式
│                   ├── 原子: 0
│                   ├── 减
│                   └── 标识符: 数
├── ，
├── 顶层表达式: 绑定
│   ├── 设
│   ├── 标识符: 结果
│   ├── 为
│   └── 表达式: 函数调用
│       ├── 标识符: 绝对值
│       ├── （
│       ├── 参数: 减性表达式
│       │   ├── 0
│       │   ├── 减
│       │   └── 3
│       └── ）
├── ，
├── 顶层表达式: 输出 → 标识符: 结果
└── 。
```

### AST 结构

```
Sentence
├── Stmt: FuncDef(
│   name="绝对值",
│   params=["数"],
│   body_type="multi",
│   body=[
│       Stmt: If(
│           condition=BinOp(left=Var("数"), op=">=", right=IntLiteral(0)),
│           then=[Stmt: ReturnResult(value=Var("数"))],
│           else=[Stmt: ReturnResult(value=BinOp(
│               left=IntLiteral(0), op="-", right=Var("数")
│           ))]
│       )
│   ]
│ )
├── Stmt: Bind(name="结果", value=Call(
│   func=Var("绝对值"),
│   args=[BinOp(left=IntLiteral(0), op="-", right=IntLiteral(3))]
│ ))
├── Stmt: Print(value=Var("结果"))
```

### 歧义评估

无。关键确认：`设绝对值（数），如果…` 中 `，` 是复式函数的函数头与函数体之间的分隔，不是句内顶层表达式分隔。解析规则：`设 标识符（形参），` 之后的内容按函数体解析。

### Python 输出

```python
def 绝对值(数):
    if 数 >= 0:
        return 数
    else:
        return 0 - 数
结果 = 绝对值(0 - 3)
print(结果)
```

## 7. 没有值

### 源码

```
设查询结果没有值，如果查询结果没有值，就显示【暂无结果】。
```

### 句法分解

```
句子
├── 顶层表达式: 绑定 (空值)
│   ├── 设
│   ├── 标识符: 查询结果
│   └── 没有值     ← 特殊绑定模式
├── ，
├── 顶层表达式: 如果句
│   ├── 如果
│   ├── 条件: 成员表达式 + 空值后缀
│   │   ├── 标识符: 查询结果
│   │   └── 没有值     ← 空值后缀判断
│   ├── ，就
│   └── 动作: 输出 → 文本: 【暂无结果】
└── 。
```

### AST 结构

```
Sentence
├── Stmt: BindNone(name="查询结果")
├── Stmt: If(
│   condition=BinOp(left=Var("查询结果"), op="is", right=NoneLiteral()),
│   then=[Stmt: Print(value=StrLiteral("暂无结果"))]
│ )
```

### 歧义评估

无。`设标识符没有值` 作为特殊绑定模式，与 `如果标识符没有值` 中空值后缀在句法上区分：前者的 `没有值` 出现在 `设` 之后的标识符右方、无 `为` 关键字；后者的 `没有值` 出现在条件表达式最后一个原子位置。

### Python 输出

```python
查询结果 = None
if 查询结果 is None:
    print("暂无结果")
```

## 9. 跳出循环

### 源码

```
设名单为［【张三】、【李四】、【王五】］，从名单中，每取一项记作姓名，就（如果姓名等于【李四】，就显示【找到了】，并跳出循环，不然就继续下一轮）。
```

### 句法分解

```
句子
├── 顶层表达式: 绑定
│   └── 名单 = ［张三、李四、王五］
├── ，
├── 顶层表达式: 遍历句
│   ├── 从名单中，每取一项记作姓名，就
│   └── 动作: 分组（）
│       └── 动作链:
│           ├── 动作: 如果句
│           │   ├── 如果 姓名等于【李四】，就
│           │   │   └── 动作链:
│           │   │       ├── 动作: 输出 → 【找到了】
│           │   │       └── 动作: 跳出循环
│           │   └── ，不然就
│           │       └── 动作: 继续下一轮
│           └── (注意：这里只有一个如果句，无后续并列动作)
└── 。
```

### AST 结构

```
Sentence
├── Stmt: Bind(name="名单", value=ListLiteral([
│   StrLiteral("张三"), StrLiteral("李四"), StrLiteral("王五")
│ ]))
├── Stmt: For(
│   target="姓名",
│   iter=Var("名单"),
│   body=[
│       Stmt: If(
│           condition=BinOp(left=Var("姓名"), op="==", right=StrLiteral("李四")),
│           then=[
│               Stmt: Print(value=StrLiteral("找到了")),
│               Stmt: Break()
│           ],
│           else=[Stmt: Continue()]
│       )
│   ]
│ )
```

### 歧义评估

无全局面歧义。需要确认：`（如果姓名等于【李四】，就显示【找到了】，并跳出循环，不然就继续下一轮）` 中 `，并跳出循环` 的归属——`并跳出循环` 是 `显示【找到了】` 后的下一个动作（在 then 分支内）。解析器应把 `并跳出循环` 解析为 then 分支的第二个动作（`并` 是动作前缀，不改变分支归属）。

关键确认：`并` 优先级不高于 `，不然就`。当看到 `，不然就` 时，当前分支结束。

### Python 输出

```python
名单 = ["张三", "李四", "王五"]
for 姓名 in 名单:
    if 姓名 == "李四":
        print("找到了")
        break
    else:
        continue
```

## 10. 异常处理

### 源码

```
尝试读取不存在文件（【missing.txt】），如果出错，就显示【读取失败】。
```

### 句法分解

```
句子
├── 顶层表达式: 尝试句
│   ├── 尝试
│   ├── 动作: 分组（）
│   │   └── 函数调用: 读取不存在文件（【missing.txt】）
│   │       ├── 标识符: 读取不存在文件
│   │       ├── （
│   │       └── 参数: 文本: 【missing.txt】
│   │       └── ）
│   ├── ，如果出错，就
│   └── 动作: 输出 → 文本: 【读取失败】
└── 。
```

### AST 结构

```
Sentence
├── Stmt: Try(
│   body=[Stmt: Call(
│       func=Var("读取不存在文件"),
│       args=[StrLiteral("missing.txt")]
│   )],
│   handler=Stmt: Print(value=StrLiteral("读取失败")),
│   err_var=None
│ )
```

### 歧义评估

无。注意 `读取不存在文件（【missing.txt】）` 是函数调用（可能函数不存在，但这是运行时错误，不是语法问题）。

### Python 输出

```python
try:
    读取不存在文件("missing.txt")
except:
    print("读取失败")
```
