# 周道 v0.0.1 实验报告

## 小结

周道是一个中文句法式 Python 方言。它不使用英文关键字、缩进、冒号或花括号，而是使用中文短语、数学表达和中文标点来表达程序。源码先解析为独立 AST，再转译为标准 Python，由 CPython 执行。

v0.0.1 实现了完整的最小可行原型：词法分析器 → 递归下降解析器 → AST → Python 发射器 → CLI。

## 样例与生成的 Python

### 倒计数
```
设数量为3，当数量大于0时，一直显示数量，并使数量减1。
```
→
```python
数量 = 3
while 数量 > 0:
    print(数量)
    数量 -= 1
```
输出: `3, 2, 1`

### 年龄分支
```
设年龄为16，如果年龄不少于18，就显示【成年】，不然，如果年龄不少于12，就显示【少年】，不然就显示【儿童】。
```
→
```python
年龄 = 16
if 年龄 >= 18:
    print("成年")
elif 年龄 >= 12:
    print("少年")
else:
    print("儿童")
```
输出: `少年`

### 逐项遍历
```
设名单为［【张三】、【李四】、【王五】］，从名单中，每取一项记作姓名，就显示姓名。
```
→
```python
名单 = ["张三", "李四", "王五"]
for 姓名 in 名单:
    print(姓名)
```
输出: `张三、李四、王五`

### 单表达式函数
```
设平方（数）为数乘数，设结果为平方（5），显示结果。
```
→
```python
def 平方(数):
    return 数 * 数
结果 = 平方(5)
print(结果)
```
输出: `25`

### 多分支函数
```
设绝对值（数），如果数不少于0，就以数为所得，不然就以0减数为所得。设结果为绝对值（0减3），显示结果。
```
→
```python
def 绝对值(数):
    if 数 >= 0:
        return 数
    else:
        return 0 - 数
结果 = 绝对值(0 - 3)
print(结果)
```
输出: `3`

### 没有值
```
设查询结果没有值，如果查询结果没有值，就显示【暂无结果】。
```
→
```python
查询结果 = None
if 查询结果 is None:
    print("暂无结果")
```
输出: `暂无结果`

### 命题状态
```
设任务完成不成立，使任务完成成立，如果任务完成成立，就显示【完成】。
```
→
```python
任务完成 = False
任务完成 = True
if 任务完成:
    print("完成")
```
输出: `完成`

### 跳出循环
```
设名单为［【张三】、【李四】、【王五】］，从名单中，每取一项记作姓名，就（如果姓名等于【李四】，就显示【找到了】，并跳出循环，不然就继续下一轮）。
```
→
```python
名单 = ["张三", "李四", "王五"]
for 姓名 in 名单:
    if 姓名 == "李四":
        print("找到了")
        break
    else:
        continue
```
输出: `找到了`

## 歧义与未支持内容

### 已解决的歧义
- **没有值的双重角色**：绑定语境 `设甲没有值` → None 初始化；条件语境 `如果甲没有值` → is None 判断。由句法位置区分。
- **设的分派**：`设甲为乙`（绑定）vs `设甲（乙）为丙`（函数）vs `设甲成立`（命题）。向前看一个 token 解决。
- **逗号角色**：`，` 在控制结构内与句级通过解析器执行顺序区分——控制结构规则优先消费内部逗号。

### 未实现
- 自然语言句式：`如果用户没有登录` → 需要语义理解，暂不支持
- `成立` 的省略：`如果任务完成` 等价于 `如果任务完成成立`，v0.0.1 中要求显式 `成立`
- 标识符转义：若用户定义名为 `没有值` 的变量会与保留短语冲突
- 类、装饰器、生成器、协程、闭包（见 non-goals）

## 下一步建议

1. **文件系统执行**：`python -m 周道 examples/counter.zd` 直接运行
2. **错误信息位置**：报错已指向源码行列，但还有美化空间
3. **成立省略**：评估 `如果任务完成` 等价于 `如果任务完成成立` 的可行性
4. **"没有" 句式**：`如果用户没有登录` 的自然语言解析
5. **更丰富的标准库别名**：更多 Python 内置函数的中文别名
6. **编辑器语法高亮**：为 VS Code 等编辑器提供 TextMate 语法文件
