# 周道 — 中文句法式 Python 方言

> 用中文写 Python。不用英文关键字、不用缩进、不用冒号、不用花括号。

周道（Zhōu Dào）是一个中文编程语言原型。源码使用完整的现代中文句法、数学表达和中文标点，先解析为独立 AST，再转译到标准 Python 执行。它借用 CPython 的运行时与生态，但不创造自己的虚拟机。

## 快速开始

```bash
cd experiments/周道
pip install -e .          # 安装包（可选）

# 运行 .zd 文件
python -m 周道 examples/counter.zd

# 仅查看生成的 Python 代码
python -m 周道 examples/counter.zd --emit

# 仅检查语法
python -m 周道 examples/counter.zd --check
```

## 示例

### 倒计数
```
设数量为3，当数量 > 0 时，一直显示数量，并使数量减 1。
```
→ `3 2 1`

同时支持汉语式写法：
```
设数量为3，当数量大于0时，一直显示数量，并使数量减1。
```
→ `3 2 1`

### 年龄分支
```
设年龄为16，如果年龄 >= 18，就显示【成年】，
不然，如果年龄 >= 12，就显示【少年】，
不然就显示【儿童】。
```
→ `少年`

### 逐项遍历
```
设名单为［【张三】、【李四】、【王五】］，
从名单中，每取一项记作姓名，就显示姓名。
```
→ `张三 李四 王五`

### 函数定义
```
设平方（数）为数 * 数。
设结果为平方（5），显示结果。
```
→ `25`

### 多步骤函数
```
设绝对值（数），
如果数 >= 0，就以数为所得，
不然就以 0 - 数为所得。
设结果为绝对值（0 - 3），显示结果。
```
→ `3`

## 语法一览

| 句式 | 形式 | 说明 |
|------|------|------|
| 绑定 | `设甲为乙。` | 建立名称 |
| 空值 | `设甲没有值。` | None 初始化 |
| 命题 | `设甲成立。` / `设甲不成立。` | 布尔值 |
| 变更 | `使甲变为乙。` | 改变值 |
| 算术 | `使甲加乙。` | 增量操作（仅汉语动词） |
| 输出 | `显示甲。` | 打印 |
| 条件 | `如果甲，就乙，不然就丙。` | if/else |
| 多分支 | `如果甲，就乙，不然，如果丙，就丁，不然就戊。` | elif |
| 循环 | `当甲时，一直乙。` | while |
| 遍历 | `从甲中，每取一项记作乙，就丙。` | for |
| 函数 | `设甲（乙）为丙。` | 单式 |
| 函数 | `设甲（乙），丙，以丁为所得。` | 复式 |
| 异常 | `尝试甲，如果出错，就乙。` | try/except |
| 跳出 | `跳出循环。` | break |
| 继续 | `继续下一轮。` | continue |
| 模块 | `引入Python模块《甲》。` | import |
| 引入 | `从Python模块《甲》中引入乙。` | from import |
| 结构指称 | `其名称` | 遍历焦点领属访问 |

## 运算符

算术和比较运算符有**汉语式**和**数学符号**两种等价写法：

| 语义 | 汉语式 | 数学符号 |
|------|--------|---------|
| 加/减/乘/除 | `加` `减` `乘` `除` | `+ - * /` |
| 幂 | — | `**` |
| 整除/余（历史别名） | `整除` `余` | `//` `%` |
| 等于/不等于 | `等于` `不等于` | `= !=` |
| 大于/小于 | `大于` `小于` | `> <` |
| 不少于/不多于 | `不少于` `不多于` | `>= <=` |
| 且/或/并非 | `且` `或` `并非（…）` | — |
| 在…中/不在…中 | `在…中` `不在…中` | — |
| 就是/不是…本身 | `就是` `不是…本身` | — |

## CLI

```bash
python -m 周道                    # 显示帮助
python -m 周道 file.zd            # 运行
python -m 周道 file.zd --emit     # 仅输出 Python 代码
python -m 周道 file.zd --check    # 仅检查语法
python -m 周道 file.zd -o out.py  # 转译为 .py 文件
```

## 设计理念

- **设不翻译 Python 关键词**：不使用 `定义` 对应 `def`、`返回` 对应 `return`，而是寻找中文读者无需编程知识也能理解的完整句式
- **数学符号 + 汉语式双表层**：`+ - * / =` 等保留数学直觉，`加 减 等于` 等保留母语节奏，两者等价
- **设与使用别**：`设` 用于建立，`使` 用于改变
- **所得即结果**：`以…为所得` 表达函数结果
- **结构指称**：`其姓名` 在遍历体内指向当前元素
- **标准数据边界**：外部数据使用标准 JSON，通过 `解析JSON` / `生成JSON` 显式转换
