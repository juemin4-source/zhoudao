# 周道 — 中文句法式 Python 方言

> 用中文写 Python。不用英文关键字、不用缩进、不用冒号、不用花括号。

周道（Zhōu Dào）是一个中文编程语言原型。源码使用完整的现代中文句法、数学表达和中文标点，先解析为独立 AST，再转译到标准 Python 执行。它借用 CPython 的运行时与生态，但不创造自己的虚拟机。

## 快速开始

```bash
cd experiments/周道
pip install -e .          # 安装包（可选）

# 运行 .zd 文件
python -m 周道 examples/counter.zd

# 仅查看生成的 Python 代码
python -m 周道 examples/counter.zd --emit

# 仅检查语法
python -m 周道 examples/counter.zd --check
```

## 示例

### 倒计数
```
设数量为3，当数量大于0时，一直显示数量，并使数量减1。
```
→ `3 2 1`

### 年龄分支
```
设年龄为16，如果年龄不少于18，就显示【成年】，
不然，如果年龄不少于12，就显示【少年】，
不然就显示【儿童】。
```
→ `少年`

### 逐项遍历
```
设名单为［【张三】、【李四】、【王五】］，
从名单中，每取一项记作姓名，就显示姓名。
```
→ `张三 李四 王五`

### 函数定义
```
设平方（数）为数乘数。
设结果为平方（5），显示结果。
```
→ `25`

### 多步骤函数
```
设绝对值（数），
如果数不少于0，就以数为所得，
不然就以0减数为所得。
设结果为绝对值（0减3），显示结果。
```
→ `3`

## 语法一览

| 句式 | 形式 | 说明 |
|------|------|------|
| 绑定 | `设甲为乙。` | 建立名称 |
| 空值 | `设甲没有值。` | None 初始化 |
| 命题 | `设甲成立。` / `设甲不成立。` | 布尔值 |
| 变更 | `使甲变为乙。` | 改变值 |
| 算术 | `使甲加乙。` | 增量操作 |
| 输出 | `显示甲。` | 打印 |
| 条件 | `如果甲，就乙，不然就丙。` | if/else |
| 多分支 | `如果甲，就乙，不然，如果丙，就丁，不然就戊。` | elif |
| 循环 | `当甲时，一直乙。` | while |
| 遍历 | `从甲中，每取一项记作乙，就丙。` | for |
| 函数 | `设甲（乙）为丙。` | 单式 |
| 函数 | `设甲（乙），丙，以丁为所得。` | 复式 |
| 异常 | `尝试甲，如果出错，就乙。` | try/except |
| 跳出 | `跳出循环。` | break |
| 继续 | `继续下一轮。` | continue |
| 模块 | `引入《甲》。` | import |
| 引入 | `从《甲》中引入乙。` | from import |

## CLI

```bash
python -m 周道                    # 显示帮助
python -m 周道 file.zd            # 运行
python -m 周道 file.zd --emit     # 仅输出 Python 代码
python -m 周道 file.zd --check    # 仅检查语法
python -m 周道 file.zd -o out.py  # 转译为 .py 文件
```

## 设计理念

- **设不翻译 Python 关键词**：不使用 `定义` 对应 `def`、`返回` 对应 `return` 这种一对一的翻译，而是寻找中文读者无需编程知识也能理解的完整句式
- **数学表达优先**：`不少于`、`大于` 等中国人熟悉的数学用语
- **设与使用别**：`设` 用于建立，`使` 用于改变
- **所得即结果**：`以…为所得` 表达函数结果，来自 "所得到的值"
- **否定进入关系**：`不等于`、`不在…中`、`没有值`、`不成立`
