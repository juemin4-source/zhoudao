# 周道语言契约 v0.9

> 面向 AI 协作的确定性语言规范。
> 本契约是周道语法的正式子集描述。AI 必须遵守。

## 核心原则

1. 周道是**确定形式语言**。相同源码每次产生相同 AST。
2. 禁止自行发明句式。只使用本契约列出的结构。
3. 禁止重新拆分精确名称 `{名称}`。
4. 禁止直接把 Python 语法夹入周道源码。
5. 所有修改必须经过 `zhoudao.check` 验证。

## 语句

### 显示

```zhoudao
显示【消息】。
显示【你好，{名称}。】。
```

`{名称}` 是精确名称引用，花括号内不再分词。

### 设（变量赋值）

```zhoudao
设 变量名 = 值
设 名称 = "周道"
设 数字 = 42
设 小数 = 3.14
设 列表 = [1, 2, 3]
设 映射 = {"键": "值"}
```

### 如果（条件）

```zhoudao
如果 条件:
    语句

如果 条件:
    语句
否则:
    语句

如果 条件1:
    语句
或者如果 条件2:
    语句
否则:
    语句
```

### 当循环

```zhoudao
当 条件:
    语句
```

### 遍历

```zhoudao
遍历 元素 于 容器:
    语句
```

### 函数

```zhoudao
功能 函数名(参数1, 参数2):
    语句
    所得 表达式
```

`所得` 用于返回值。

### 引入

```zhoudao
引入 模块名
引入周道源文件《工具》。
从中 模块 引入 名称
```

## 表达式

### 字面量

| 类型 | 示例 |
|------|------|
| 整数 | `42` |
| 小数 | `3.14` |
| 文本 | `"hello"` |
| 布尔 | `真`, `假` |
| 空 | `空` |
| 列表 | `[1, 2, 3]` |
| 映射 | `{"a": 1}` |

### 运算符

| 类别 | 符号 |
|------|------|
| 算术 | `+`, `-`, `*`, `/`, `//`, `%` |
| 比较 | `==`, `!=`, `<`, `>`, `<=`, `>=` |
| 逻辑 | `且`, `或`, `非` |
| 成员 | `在`, `不在` |
| 身份 | `是`, `不是` |

### 精确名称

`{名称}` — 花括号内的内容被视为单一标识符，永不重新分词。

### 成员访问

```zhoudao
对象.属性
```

### 调用

```zhoudao
函数名(参数1, 参数2)
```

## 标点与界符

| 符号 | 作用 |
|------|------|
| `【】` | 显示字符串定界符，包含插值 `{名称}` |
| `。` | 语句终止符 |
| `：` | 块起始（中文冒号） |
| `「」` | 字符串字面量替代 |
| `""` | 字符串字面量 |
| `#` | 行注释 |
| `（` `）` | 参数列表（全角括号） |

## 已知不支持

以下结构当前不支持（AI 不得生成）：

- `match` / `分情形` 模式匹配（WIP）
- `async` / `等待` 异步（WIP）
- 生成器表达式
- 装饰器
- 类型注解
- `with` / `以所得`
- 异常捕获 `尝试/报错`
- 切片语法 `[开始:结束]`
- 类定义 `类别`
- 列表推导式
