# 周道语言契约 v0.10

> 面向 AI 协作的确定性语言规范。
> 本契约是周道语法的正式子集描述。AI 必须遵守。

## 核心原则

1. 周道是**确定形式语言**。相同源码每次产生相同 AST。
2. 禁止自行发明句式。只使用本契约列出的结构。
3. 禁止重新拆分精确名称 `{名称}`。
4. 禁止直接把 Python 语法夹入周道源码。
5. 所有修改必须经过 `zhoudao.check` 验证。

## 句子结构

- `。`（句号）结束一个句子
- `，`（逗号）分隔句子内的动作
- `：`（冒号）引入代码块（如函数体）
- 没有缩进块结构

## 赋值与状态

```zhoudao
设名称【值】                # 绑定：设名称为【值】
设名称为3。                 # 整数
设名称为【文本】。           # 文本（仅用【】）
设名称为［1，2，3］。       # 列表
设名为没有值。              # None
设任务完成成立。             # True
设任务完成不成立。           # False
```

## 变更

```zhoudao
使数量变为5。              # 重新赋值
使数量加1。                # +=
使数量减1。                # -=
使数量乘2。                # *=
使数量除2。                # /=
使任务完成成立。            # 设为 True
使任务完成不成立。          # 设为 False
```

## 显示

```zhoudao
显示【消息】。
显示名称。
```

## 条件

```zhoudao
如果数量 > 0，就显示【正数】。
如果数量 = 0，就显示【零】。
如果年龄 >= 18，就显示【成年】。
```

## 循环

```zhoudao
当数量 > 0 时，一直显示数量，并使数量减 1。
从名单中，每取一项记作姓名，就显示姓名。
```

## 函数

```zhoudao
设平方（数）为数 * 数。
设绝对值（数），如果数 >= 0，就以数为所得，不然就以 0 - 数为所得。
```

## 表达式

### 字面量

| 类型 | 写法 |
|------|------|
| 整数 | `3` `42` `-5` |
| 小数 | `3.14` |
| 文本 | `【内容】` |
| 布尔 | `真` `假` |
| 空 | `没有值` |
| 列表 | `［1，2，3］` |
| 映射 | `［【键】为【值】］` |

### 运算符

AI 默认使用符号式，除非项目配置指定 verbal：

| 语义 | 规范形式 | 示例 |
|------|---------|------|
| 加减乘除 | `+ - * /` | `数量 + 1` |
| 幂 | `**` | `甲 ** 乙` |
| 比较 | `= != < > <= >=` | `年龄 >= 18` |
| 整除/取余 | `//` `%` | `甲 // 乙` |
| 逻辑 | `且` `或` `并非（…）` | `甲且乙` |
| 成员 | `在…中` `不在…中` | `甲在乙中` |
| 身份 | `就是` `不是…本身` | `甲就是乙` |
| 成员访问 | `的` | `用户的姓名` |

### 精确名称

`{名称}` — 内容为单一标识符，永不重新分词。
用于关键字冲突时：`设{其}为值。`

### 结构指称

```zhoudao
从列表中，每取一项记作项目，就显示其名称。
```

`其` 在遍历体内指向当前遍历元素。无焦点时不得使用。

### 调用

```zhoudao
显示名称。
设结果为平方（5）。
```

### 引入

```zhoudao
引入Python模块《math》。
从Python模块《math》中引入 sqrt。
```

### JSON

```zhoudao
设数据为解析JSON（文本）。
设文本为生成JSON（值）。
```
