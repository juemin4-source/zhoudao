# 周道语言契约 v0.9

> 面向 AI 协作的确定性语言规范。
> 本契约是周道语法的正式子集描述。AI 必须遵守。

## 核心原则

1. 周道是**确定形式语言**。相同源码每次产生相同 AST。
2. **Token 之间没有空格**。所有关键字、标识符、运算符在源码中紧邻。
3. 禁止自行发明句式。只使用本契约列出的结构。
4. 禁止重新拆分精确名称 `{名称}`。
5. 禁止直接把 Python 语法夹入周道源码。
6. 所有修改必须经过 `zhoudao.check` 验证。

## 句子结构

- `。`（句号）结束一个句子
- `，`（逗号）分隔句子内的动作
- `：`（冒号）引入代码块（如函数体）
- 没有缩进块结构

## 赋值与状态

```zhoudao
设名称【值】                # 绑定：设名称为【值】
设名称为3。                 # 整数
设名称为【文本】。           # 文本（仅用【】）
设名称为［1，2，3］。       # 列表
设名为没有值。              # None
设任务完成成立。             # True
设任务完成不成立。           # False
```

## 变更

```zhoudao
使数量变为5。              # 重新赋值
使数量加1。                # +=
使数量减1。                # -=
使数量乘2。                # *=
使数量除2。                # /=
使任务完成成立。            # 设为 True
使任务完成不成立。          # 设为 False
```

## 显示

```zhoudao
显示【消息】。              # print("消息")
显示名称。                  # print(名称) — 变量名无需【】
显示【{名称}】。            # 精确名称插值
```

## 条件

```zhoudao
如果条件，就动作。
如果数量大于0，就显示【正数】。
如果数量大于0，就显示【正数】，不然就显示【非正数】。
如果数量大于0，就显示【正数】，不然，如果数量等于0，就显示【零】，不然就显示【负数】。
```

## 循环

```zhoudao
当条件时，一直动作。
当数量大于0时，一直显示数量，并使数量减1。

从集合中，每取一项记作元素名，就动作。
从名单中，每取一项记作姓名，就显示姓名。
```

## 函数

单表达式函数：

```zhoudao
设名称（参数）为表达式。
设平方（数）为数乘数。
```

多语句函数：

```zhoudao
设名称（参数），如下：动作。
设绝对值（数），如下：如果数不少于0，就以数为所得，不然就以0减数为所得。
```

`以...为所得` 或 `以...所得` 表示返回值。

## 列表中括号分组

```zhoudao
（动作1，动作2）。
```

用 `（）` 分组多个动作，如：

```zhoudao
如果姓名等于【李四】，就（显示【找到了】，并跳出循环），不然就继续下一轮。
```

## 表达式

### 字面量

| 类型 | 写法 |
|------|------|
| 整数 | `3` `42` `-5` |
| 小数 | `3.14` |
| 文本 | `【内容】` |
| 布尔 | `真` `假` |
| 空 | `没有值` |
| 列表 | `［1，2，3］` `［【a】、【b】］` |
| 映射 | `{【键】: 【值】}` |

### 运算符

| 类别 | 写法 |
|------|------|
| 加 | `x加y` |
| 减 | `x减y` |
| 乘 | `x乘y` |
| 除 | `x除y` |
| 整除 | `x整除y` |
| 余 | `x余y` |
| 等于 | `x等于y` |
| 不等于 | `x不等于y` |
| 大于 | `x大于y` |
| 小于 | `x小于y` |
| 不少于 | `x不少于y` |
| 不多于 | `x不多于y` |
| 且 | `x且y` |
| 或 | `x或y` |
| 并非 | `并非x` |
| 在 | `x在y中` |
| 不在 | `x不在y中` |
| 的（成员） | `x的y`（如 `用户的姓名`） |

### 精确名称

`{名称}` — 花括号内的内容被视为单一标识符，永不重新分词。
示例：`显示【你好，{用户的姓名}。】。`

### 调用

```zhoudao
显示名称。                # 无参调用：显示 变量名
设结果为平方（5）。        # 函数调用：名称（参数）
```

## 引入

```zhoudao
引入《模块名》。          # import module
从《模块》中引入名称。    # from module import name
```

## 成员标记

```zhoudao
下文所用甲，均指全局的甲。    # global
下文所用甲，指本定义外层的甲。  # nonlocal
```

## 已知不支持

以下结构当前不支持（AI 不得生成）：

- `类别` 与面向对象（WIP）
- `尝试`/`如果出错` 异常捕获
- `依次给出` 生成器
- `等待`/`完成` 异步
- `依`/`分情形` 模式匹配
- 装饰器
- 类型注解
- 切片语法
- 列表推导式
