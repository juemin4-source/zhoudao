# 函数定义与参数语义

> 隶属：周道语言规范 1.0 — 第三层：语义规则

---

## 1. 默认参数求值时机

### 1.1 规则

默认参数在**函数定义时求值一次**，而非每次调用时重新求值。

```
定义函数（项目们默认为［］）如下：
```

`［］` 在定义时创建，所有对该函数的调用共享同一列表对象。

### 1.2 可变容器警告

当默认参数为可变容器（列表、集合、映射）时，发出诊断：

```text
警告：可变容器「［］」被用作默认参数，多个调用将共享同一对象。
```

### 1.3 诊断级别

```toml
[diagnostics]
mutable_default = "warning"
```

## 2. 函数结束语义

### 2.1 无显式所得

函数自然结束时（不执行 `以…为所得`），返回 `没有值`：

```
定义记录日志（内容）如下：
显示内容。
```

等价于 Python：

```python
def 记录日志(内容):
    print(内容)
    return None
```

### 2.2 生成器结束

生成器自然结束时，StopIteration 不携带值。

### 2.3 显式空返回

`以没有值为所得。` 显式返回空值，与自然结束一致。

---

## 3. 独立调用语句

函数调用本身可以作为独立语句执行，不需要绑定结果：

```
保存文件（）。
发送消息（内容）。
任务的启动（）。
```

这在 Python 中对应：

```python
save_file()
send_message(content)
task.start()
```

生成的 Python 代码为表达式语句，不赋值。此类调用的返回值被丢弃。
